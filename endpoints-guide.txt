===========================
 Kid Buu Leaderboard Backend - API Endpoints
===========================

üåê Base URL (local):
http://localhost:3000

---

1) /leaderboard
- URL: http://localhost:3000/leaderboard
- Returns: JSON array of the top 250 holders
- Fields: rank, wallet, balance, pct, multiplier, entries, currentValue, formRequired

---

2) /myentries?wallet=<WALLET_ADDRESS>
- URL Example: http://localhost:3000/myentries?wallet=YourWalletPubkey
- Returns: JSON object with stats for a single wallet
- Use case: Called by the MiniAppCard after connect-wallet

---

3) /wallet/:address
- URL Example: http://localhost:3000/wallet/YourWalletPubkey
- Returns: Same as /myentries but REST-style
- Use case: Manual lookup for debugging (you don‚Äôt need a frontend wallet connect)

---

4) /price
- URL: http://localhost:3000/price
- Returns: { token, price, lastUpdated }
- Price is fetched from Jupiter API (cached between leaderboard updates)

---

5) /download
- URL: http://localhost:3000/download
- Returns: A downloadable PDF snapshot of the current leaderboard
- Contains: Rank, Wallet, Balance, % Supply, Multiplier, Entries, Current Value, Form Required

---

üí° Notes:
